# Quickstart

```php
<?php

use Guozhu\Ch4\Context;
use Guozhu\Ch4\HTTPMethod;
use Guozhu\Ch4\Router;

include 'vendor/autoload.php';

$engine = new \Guozhu\Ch4\Engine();

Router::addRouter(HTTPMethod::Get, '/', function (Context $context) {
    echo "hello world!";
});

$engine->run();
```
